Java OOP Practice

Repositório dedicado aos exercícios que estou desenvolvendo para praticar Programação Orientada a Objetos (POO) em Java, durante meus estudos em desenvolvimento backend.

O objetivo deste repositório é consolidar conceitos fundamentais da linguagem Java e fortalecer a base necessária para o desenvolvimento de aplicações backend.

Conteúdos praticados

Os exercícios deste repositório abordam principalmente:

Programação Orientada a Objetos (POO)

Herança

Polimorfismo

Encapsulamento

Classes e objetos

Estrutura de projetos em Java

Lógica de programação aplicada

Estrutura do repositório

Cada exercício é desenvolvido em um arquivo ou pasta separada para facilitar a organização e o entendimento.

Exemplo de estrutura:

java-oop-exercises
│
├── exercicio1-animal
├── exercicio2-funcionarios
├── exercicio3-pagamentos

Exercícios desenvolvidos
Exercício 1 – Animais

Prática de herança e polimorfismo com diferentes tipos de animais.

Exercício 2 – Funcionários

Simulação de diferentes tipos de funcionários utilizando sobrescrita de métodos para cálculo de salário.

Exercício 3 – Sistema de Pagamentos

Implementação de diferentes formas de pagamento utilizando polimorfismo.
